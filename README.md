# 🧠 DP-Masters: Behavioral Psychology Learning Platform

DP-Masters is a comprehensive online education platform designed to teach the nuances of human behavior, dark psychology, and conversational hypnosis. This project bridges the gap between complex psychological concepts and real-world application.

## ✨ Key Features
* **User Authentication:** Secure login and registration using **Firebase Auth** (including Google Sign-In).
* **Lead Generation:** Dedicated registration flows for capturing potential student interests.
* **Secure Payments:** Integrated **Razorpay** gateway for subscription-based access.
* **Responsive UI:** Built with modern CSS/HTML for a seamless experience across devices.

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS3, HTML5
* **Backend:** Node.js, Express.js
* **Database & Auth:** Firebase
* **Payment Gateway:** Razorpay
* **Security:** SHA-256 Password Hashing & Salted Integrity
